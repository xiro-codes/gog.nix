# GOG.nix

A Nix flake for managing and installing GOG (Good Old Games) library on NixOS and other Linux distributions using Nix.

## Features

- **Automated Packaging**: Includes a Python script to scan installer directories and generate Nix expressions automatically.
- **Custom Fetcher**: Uses a specialized fetcher to retrieve game installers from a local or private file server.
- **Native & Windows Support**: Provides abstractions for both native Linux GOG installers and Windows-based games via Wine/Proton.

## Project Structure

- `pkgs/`: Contains individual game definitions generated by the scanning tool.
- `libs/`: Core logic for building game packages and fetching assets.
- `generate_pkgs.py`: A utility to hash installers and create new package directories.

## Getting Started

### 1. Generate Packages
Point the generator script at your directory of GOG `.sh` installers:
```bash
python3 generate_pkgs.py /path/to/my/gog/library -o pkgs/games
```
